### How to use it
+ Set up your own datasource in application.properties
+ Use spring.jpa.hibernate.ddl-auto=update for updating the database, otherwise it wont update and wont do anything
+ Run the ProjectApplication
+ Go to a browser
+ Access Swagger-UI : http://localhost:8080/swagger-ui/index.html#/
+ View all the API Endpoints and how to use the coresponding services
